## How to run locally

* Download this repo or run: 
```bash
    $ git clone git@github.com:kayprogrammer/booking_app.git
```

#### In the root directory:
- Install Go
```bash
    $ sudo apt update && sudo apt upgrade && sudo apt install golang
```
- Build and run server
```bash
    $ go run main.go
```
OR
```bash
    $ go run .
```
